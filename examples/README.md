# Java examples

This is a Java (Gradle) project configured to compile and test the generated
examples. It targets Java 17 and depends on JSpecify for nullness annotations.

The Java sources under `src/main/java` are generated by `sqlc-gen-java` from the
schemas and queries under `src/main/resources`. Regenerate them with:

```shell script
make -C .. all      # build bin/sqlc-gen-java.wasm
sqlc generate       # run from this directory
```

To run the tests you need Postgres and MySQL running locally:

```shell script
docker run --name dinosql-postgres -d -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=mysecretpassword -e POSTGRES_DB=postgres -p 5432:5432 postgres:11
docker run --name dinosql-mysql -d -e MYSQL_ROOT_PASSWORD=mysecretpassword -e MYSQL_DATABASE=mysql -p 3306:3306 mysql:8
./gradlew clean test
```

The project can be easily imported into IntelliJ IDEA:

1. Install a JDK 17+ if you don't already have one
1. Download IntelliJ IDEA Community Edition
1. In the "Welcome" modal, click "Import Project"
1. Open the `build.gradle` file adjacent to this README file
1. Wait for IntelliJ to sync the gradle modules and complete indexing
